Redirect Manager

Anleitung

Schritt 1: URL hinzufügen (nur mit Authentifizierung)

  1. Öffnen Sie den Redirect Manager in Ihrem Webbrowser.
  2. Navigieren Sie zum Abschnitt "Einträge hinzufügen".
  3. Geben Sie den gewünschten Slug und die entsprechende URL in die entsprechenden Eingabefelder ein.
  4. Klicken Sie auf die Schaltfläche "Submit".

Schritt 2: URL aufrufen

  1. Öffnen Sie Ihren Webbrowser.
  2. Geben Sie die Haupt-URL des Redirect Managers gefolgt vom gewünschten Slug in die Adressleiste ein. Beispiel: https://localhost:3000/:slug (ersetzen Sie :slug durch den gewünschten Slug).
  3. Drücken Sie die Eingabetaste oder klicken Sie auf "Los".
  4. Sie werden automatisch zur entsprechenden URL weitergeleitet.

Schritt 3: Alle Einträge anzeigen

  1. Authentifizieren Sie sich als Administrator oder mit den erforderlichen Zugangsdaten.
  2. Navigieren Sie zum Endpunkt "/entries".
  3. Die Liste aller Einträge wird angezeigt, einschließlich des Slugs und der entsprechenden URLs.

Schritt 4: Eintrag löschen (nur mit Authentifizierung)

  1. Authentifizieren Sie sich als Administrator oder mit den erforderlichen Zugangsdaten.
  2. Navigieren Sie zum Endpunkt "/entry/:slug" (ersetzen Sie :slug durch den zu löschenden Slug).
  3. Der Eintrag mit dem angegebenen Slug wird aus der Liste entfernt und die URL ist nicht mehr verfügbar.

Schritt 5: Automatisches Generieren eines Slugs (optional)

Wenn Sie beim Hinzufügen einer URL keinen spezifischen Slug angeben, wird automatisch ein zufälliger Slug generiert. Der generierte Slug wird zusammen mit der URL gespeichert und kann für den Zugriff auf die URL verwendet werden.

Bitte beachten Sie, dass für bestimmte Endpunkte eine Authentifizierung erforderlich ist, um die Aktionen auszuführen. Stellen Sie sicher, dass Sie über die entsprechenden Berechtigungen verfügen, um auf diese Endpunkte zugreifen zu können.
